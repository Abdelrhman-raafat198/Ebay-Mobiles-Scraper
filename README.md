## Overview
This project is a web scraping tool designed to collect mobile phone data from eBay. It automates the process of searching for specific models and extracts key information into a structured format.

## Features
- **Dynamic Scraping:** Uses **Selenium** to handle dynamic content and pagination.
- **Data Structuring:** Uses **Pandas** to organize data into DataFrames.
- **Export:** Saves the final results into a `csv` file for easy analysis.

## Tech Stack
- **Language:** Python
- **Libraries:** Selenium, Pandas, Webdriver-manager

## How to Run
1. Clone the repo: `git clone https://github.com/your-username/ebay-mobiles-scraper.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python scraper.py`
